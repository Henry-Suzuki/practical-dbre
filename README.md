## ディレクトリ構造

```
research/
  companies/         # Companyの情報
    google/          # Googleの情報
    microsoft/       # Microsoftの情報
    amazon/          # Amazonの情報
    netflix/         # Netflixの情報
    circleci/        # CircleCIの情報
    spotify/         # Spotifyの情報
  comparisons/       # 比較分析
```

## 各社の情報カテゴリ

各社のディレクトリには以下の情報を収集：

1. **career-ladder.md** - エンジニアリングラダー（キャリアラダー）
2. **org-structure.md** - 組織構造とチーム設計
3. **performance-review.md** - 評価とパフォーマンスマネジメント
4. **hiring.md** - 採用とオンボーディング
5. **culture.md** - エンジニアリング文化とプラクティス

## 情報収集の方針

- 各社が公開している情報を優先
- 公式ブログ、GitHub、公開資料など
- 情報の出典を必ず記載
- 情報の更新日を記載
- 情報の信頼性を評価
